## Project Overview

Pravoslavni Citati ("Orthodox Quotes") is a Serbian Orthodox Christian daily Bible verse app built with **Expo + React Native** (frontend) and **Express.js** (backend). It includes a **native iOS WidgetKit extension** for lock screen widgets.

All UI text and Bible verses are in **Serbian Cyrillic**.

## Tech Stack

- **Frontend:** Expo SDK 54, React Native 0.81, React 19, Expo Router 6, TypeScript
- **Backend:** Express 5, PostgreSQL, Drizzle ORM
- **iOS Widget:** SwiftUI + WidgetKit (targets/bible-widget/index.swift)
- **Animations:** react-native-reanimated, expo-haptics
- **Data fetching:** TanStack React Query
- **Native targets:** @bacons/apple-targets for iOS widget integration

## Common Commands

```bash
# Frontend dev server
npm run expo:dev

# Backend dev server
npm run server:dev

# Static production build (Expo)
npm run expo:static:build

# Backend production build
npm run server:build

# Push database schema changes
npm run db:push
```

iOS widget requires **Expo EAS build** for production deployment (cannot be tested in Expo Go).

## Architecture

### App Screens (Expo Router - file-based routing)
- `app/index.tsx` — Home screen: live clock, date, daily verse carousel
- `app/widget-info.tsx` — Modal with widget setup instructions
- `app/_layout.tsx` — Root layout: fonts, QueryClient, error boundary

### iOS Widget
- `targets/bible-widget/index.swift` — WidgetKit extension with 37 hardcoded Serbian Bible verses
- Selects daily verse by `dayOfYear % count`; timeline refreshes at midnight
- Supports systemSmall, systemMedium, systemLarge families
- Uses `glassEffect()` on iOS 26.0+, `containerBackground` on iOS 17.0+

### Backend
- `server/index.ts` — Express server, serves static Expo manifests and API
- `server/routes.ts` — API route registration (template)
- `shared/schema.ts` — Drizzle ORM schema (users table)

### Shared Code
- `constants/colors.ts` — Theme: gold (#C9A84C) accent, deep navy (#080C18) background
- `lib/query-client.ts` — API client with TanStack Query; base URL from `EXPO_PUBLIC_DOMAIN`
- `components/` — ErrorBoundary, ErrorFallback, KeyboardAwareScrollViewCompat

## Key Details

- TypeScript path aliases: `@/*` maps to project root, `@shared/*` maps to `shared/`
- No i18n library — all strings are hardcoded in Serbian Cyrillic
- Widget verses are duplicated: hardcoded in Swift (widget) and in the React Native app
- The `ios/` and `android/` directories are gitignored — generated by Expo prebuild/EAS
- Deployment configured for Replit CloudRun (see `.replit`)
